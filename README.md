# Dice Catcher

A small 2D arcade game made with Godot 4.6 while studying the course:
"Jumpstart to 2D Game Development with Godot 4.6".

The player controls a fox and catches falling dice to score points and survive as long as possible.

## Gameplay

- Normal Dice → +1 point
- Golden Dice → +5 points
- Bad Dice → lose 1 life when caught
- Missing normal/golden dice → lose 1 life
- Missing bad dice → no penalty
- Gain +1 life every 10 points (up to 3 lives)
- Difficulty increases over time

## Features

- Start screen
- Game over screen
- Persistent high score
- Combo streak system
- Difficulty progression
- Floating feedback labels
- Particle effects
- Screen shake
- Sound effects and music
- Scene transitions/loading scene
- Exported builds for Web, Windows, and Linux
